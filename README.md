# 🚀 ForgeRAG

ForgeRAG is a fullstack Retrieval-Augmented Generation (RAG) application combining a modern web interface, a secure backend, and a local AI pipeline.

It allows users to ask questions on their own documents and receive contextualized answers generated by a local LLM, with real-time streaming and source transparency.

---

## ✨ Features

### 🔐 Authentication

* User registration & login (JWT-based)
* Secure API access
* Token stored on frontend (localStorage)

### 🤖 RAG Pipeline

* Document ingestion (PDF, DOCX, XML)
* Text chunking & embedding
* Vector search using ChromaDB
* Context retrieval from relevant chunks

### 🧠 AI Generation

* Local LLM via LM Studio
* Prompt-based contextual answering
* Streaming responses (token-by-token)

### 💬 Frontend UI

* React + Vite + Tailwind + shadcn/ui
* ChatGPT-like interface
* Real-time streaming display
* Source visualization (accordion + excerpts)

### 📜 History

* User conversation history stored in database
* Retrieved and displayed on reload

## ⚙️ Setup

### 1. Clone the repository

```bash
git clone <repo-url>
cd ForgeRAG
```

---

### 2. Start the application

```bash
docker compose up --build
```

---

### 3. Access the services

| Service     | URL                   |
| ----------- | --------------------- |
| Frontend    | http://localhost:5173 |
| Backend API | http://localhost:3000 |
| RAG Service | http://localhost:8000 |

---

### 4. LM Studio Setup

* Launch LM Studio
* Load a model (e.g. `qwen/qwen3-vl-4b`)
* Enable:

  * API Server
  * "Serve on local network"

---

## 🔑 Environment Variables

Example `.env`:

```env
DATABASE_URL=postgresql://user:password@db:5432/forgerag
OPENAI_BASE_URL=http://host.docker.internal:1234/v1
OPENAI_API_KEY=lm-studio
LLM_MODEL=qwen/qwen3-vl-4b
RAG_SERVICE_URL=http://rag-service:8000
```
